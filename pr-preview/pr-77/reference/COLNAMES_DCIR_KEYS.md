# Neufs clés de jointure du DCIR

Neufs clés de jointure du DCIR

## Usage

``` r
COLNAMES_DCIR_KEYS
```

## Format

An object of class `character` of length 9.

## Value

Character vector. Neufs clés de jointure du DCIR : table principale
ER_PRS_F et [tables
affinées](https://documentation-snds.health-data-hub.fr/snds/fiches/tables_affinees.html#les-tables-affinees#les-tables-affinees).
