# Environnement d'exécution de sndsTools

Environnement d'exécution de sndsTools

## Usage

``` r
IS_PORTAIL
```

## Format

An object of class `logical` of length 1.

## Value

Boolean. True si le package sndsTools est exécuté dans l'environnement
de travail du Portail, False sinon.
