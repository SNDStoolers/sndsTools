# Nom de la table temporaire pour les filtres ir_pha_r

Nom de la table temporaire pour les filtres ir_pha_r

## Usage

``` r
TNAME_FILTER_IR_PHA_R
```

## Format

An object of class `character` of length 1.

## Value

A string with the name of the temporary table for ir_pha_r filters.
