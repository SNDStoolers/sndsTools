# Nom de la table temporaire pour les IDs de patients

Nom de la table temporaire pour les IDs de patients

## Usage

``` r
TNAME_FILTER_PATIENTS
```

## Format

An object of class `character` of length 1.

## Value

A string with the name of the temporary table for patient IDs.
