# Insertion des résultats d'une requête SQL dans une table existante.

Insertion des résultats d'une requête SQL dans une table existante.

## Usage

``` r
insert_into_table_from_query(
  conn = NULL,
  output_table_name = NULL,
  query = NULL
)
```

## Arguments

- conn:

  Connexion à la base de données

- output_table_name:

  Nom de la table de sortie

- query:

  Requête SQL
